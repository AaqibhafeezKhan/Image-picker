ImagePicker

Android library to pick images from Gallery and Camera with auto-rotation, cropping, and compression.

Installation

Add JitPack to project build.gradle:

maven { url 'https://jitpack.io' }

Add dependency to app build.gradle:

implementation 'com.github.AaqibhafeezKhan:Image-picker:0.1.2'

Usage

ImagePicker imagePicker = new ImagePicker(this, onFileChoose);
imagePicker.requestImageCamera(101, true, true);
