ImagePicker

Android library to pick images from Gallery and camera. Includes rotation handling (EXIF), cropping, and compression.

Features

  • Easy implementation
  • No manual runtime permissions boilerplate
  • Image rotation handling using EXIF
  • Built-in cropping and compression
  • Native file path handling

Installation

Add JitPack to project build.gradle:

allprojects {
    repositories {
        maven { url 'https://jitpack.io' }
    }
}

Add dependency to app build.gradle:

implementation 'com.github.AaqibhafeezKhan:Image-picker:0.1.2'

Usage

1. Initialize

ImagePicker imagePicker = new ImagePicker(this, onFileChoose);
ImageCallBackManager callBackManager = imagePicker.getCallBackManager();

2. Callback

private ImagePicker.OnFileChoose onFileChoose = new ImagePicker.OnFileChoose() { 
    @Override 
    public void onFileChoose(String fileUri, int requestCode) { 
        // Image path received here
    } 
};

3. Forward Results

@Override 
public void onRequestPermissionsResult(...) { 
    callBackManager.onRequestPermissionsResult(...); 
}

@Override 
protected void onActivityResult(...) { 
    callBackManager.onActivityResult(...); 
}

4. Open Picker

imagePicker.requestImageCamera(PERMISSION_CODE, true, true); 
imagePicker.requestImageGallery(PERMISSION_CODE, true, true);
